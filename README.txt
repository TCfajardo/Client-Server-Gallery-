Proyecto Cliente-Servidor para Envío y Visualización de Imágenes

- Descripción del proyecto:

Este proyecto implementa una aplicación cliente-servidor para enviar imágenes desde el cliente al servidor
y visualizar una galería de imágenes en el cliente. El objetivo principal es permitir a los usuarios cargar
imágenes desde su aplicación cliente y visualizarlas en una galería proporcionada por el servidor.

- Instrucciones de instalación:

Clona o descarga el repositorio del proyecto desde https://github.com/TCfajardo/Client-Server-Gallery-.git.
Abre el proyecto en tu entorno de desarrollo preferido (por ejemplo VSC, Eclipse, IntelliJ, NetBeans, etc.).
Compila y ejecuta el servidor en tu máquina.
Compila y ejecuta el cliente en los dispositivos desde los que deseas cargar imágenes y ver la galería. tanto
el cliente como el servidor deben estar en la misma red.

Requisitos del sistema:

Java Development Kit (JDK) versión 8 o superior.
Conexión a la misma red local para la comunicación entre el cliente y el servidor.
Sistema operativo compatible con Java (Windows, Linux, macOS, etc.).
